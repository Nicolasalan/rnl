# Web Documentation Setup Guide
To initiate the web documentation, follow the steps below:

### Step 1: Clone the Repository

<CodeGroup>
```bash {{ title: 'bash' }}
git clone https://github.com/microvault/microvault.git
cd microvault/docs
```
</CodeGroup> 

### Step 2: Set Up the Environment
Execute the following command to set up the environment using yarn:

<CodeGroup>
```bash {{ title: 'bash' }}
yarn
```
</CodeGroup>

### Step 3: Start the Documentation App
Once the environment setup is complete, start the documentation app using the following command:
<CodeGroup>
```bash {{ title: 'bash' }}
yarn dev
```
</CodeGroup>
