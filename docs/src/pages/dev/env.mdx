# Microvault Setup Guide
To install all system dependencies and start working on Microvault, follow these steps:

### Step 1: Clone the Repository
```bash
git clone https://github.com/microvault/microvault.git
```

### Step 2: Install Poetry
```bash
pip install poetry
```
### Step 3: Navigate to the Microvault Repository
```bash
cd microvault
```
### Step 4: Install Project Dependencies
```bash 
poetry install
```
This will set up the project environment and install the required dependencies.
### Step 5: Install Dev Dependencies
```bash
poetry install --dev
```
After installing all dependencies, you are ready to start working on Microvault.
